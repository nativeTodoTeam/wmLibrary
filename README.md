# wmLibrary
蛙鸣图书馆
## 启动server
* npm start

## 启动App
* wmApp目录下 ios端:react-native run-ios  android端:react-native run-android

## 接口文档
* npm run apidoc
* 项目启动后访问: http://localhost:3000/apidoc/index.html

## 项目结构
* bin 启动目录
* config 配置文件
* logs 日志文件
* models 与数据库表一一对应
* routes -> api 接口目录
* utils 格式文件
* views 页面文件
* views Native app
